# Job-Post-Scraping
Python script that scrapes Job posting data from a website.

* Python v3.7

## Modules Used
* Pandas
* bs4
* Selenium WebDriver
* openpyxl

## Installing the modules
* Installing Pandas 
```pip install pandas```

* Installing BeautifulSoup 
```pip install bs4```

* Installing selenium 
```pip install selenium```

* Installing openpyxl 
```pip install openpyxl```
